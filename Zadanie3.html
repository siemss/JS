3) Omów typy zmiennych w Javascript:

var - typ automatyczny (można przypisać dowolny typ)

null - Nie ma nic

undefined - niezainicjowane zmienne

boolean - true or false

string - tekst

number - liczba

object - Obiekt

symbol - symbol

Przy nadawaniu nazw zmiennym należy przestrzegać pewnych zasad(trzy zasady):

Nazwa zmiennej powinna jak najdokładniej odpowiadać przechowywanym w niej danym.

Bez transliteracji. Tylko angielski.

Nazwy zmiennych nie mogą zawierać spacji.
